# Aaron's 22nd Birthday Book

A static HTML/CSS/JavaScript flip-book style website with 22 letters.

## Files

- `index.html` - the webpage
- `styles.css` - the scrapbook/book styling
- `script.js` - the page flipping and 22 letter content

## Edit the letters

Open `script.js` and replace the placeholder text inside the `letters` array.

## Deploy on Cloudflare Pages

Use these settings:

- Framework preset: None
- Build command: leave blank
- Build output directory: `/`
